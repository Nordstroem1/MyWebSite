Bigger projects

I've worked on several projects that have helped me develop my skills and gain experience in building robust and scalable applications. Here are some of the projects.

BoostApp

The boostapp was an internal application I took part in refactor and develop using .NET8 along with hosting on Azure Portal the applications purpose was to help the company empolees to complete challenges as report working hours in time, completing team challenges as taking 10 000 steps each day, doing yoga together etc etc.

During the applications development we used Azure devops for CI-CD, it was built with a microservice architecture. For the diffrent apis to comunicate we used Azure servicebus witch was fun to learn. other features was wokring with diffrent databases as cosmos db along with SQL in the many apis.

EmailBuilder

This project is an internal email sender. The company structure is built on three levels: Business Area, Business Unit, and Competence Unit. The application is role-based, for example ensuring that employees at the Competence Unit level cannot see emails from higher levels. To fetch users and groups, I connected the application's API to Microsoft's Graph API to access Azure Entra ID.

I had the opportunity to build the backend from scratch using Azure DevOps for CI/CD, employing design patterns such as clean architecture and generic repository. To reduce costs, the team decided to use NoSQL with MongoDB, which was both fun and necessary to learn. The project is hosted a Azure portal.
